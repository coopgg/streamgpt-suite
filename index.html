Get set up in minutes
You need Streamerbot, Python 3, and an OpenAI API key.
1

Download the installer

Get StreamGPT_Suite_Setup_v0.9-beta.py. This single file contains the entire suite.

2

Run the installer

Open a terminal and run python StreamGPT_Suite_Setup_v0.9-beta.py. Answer the prompts — your API key, streamer name, currency name, emotes, and which systems to install.

3

Import into Streamerbot

Open Streamerbot → Import → select StreamGPT_Suite_Import.json. All 124 actions are created automatically.

4

Compile and go live

Open each Execute - #Code action and click Save and Compile. Type !balance in chat — if the bot responds, you're live.
