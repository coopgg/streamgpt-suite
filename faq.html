Troubleshooting

Start with these in order:

  • Check the Streamerbot action log — it shows exactly which sub-actions ran and what failed
  • Make sure you clicked Save and Compile on every Execute - #Code action in Streamerbot
  • Confirm your OpenAI API key is correct and your account has billing enabled
  • Check that C:\StreamGPT exists and the scripts are in the right subfolders

The setup guide includes a troubleshooting section with the most common issues and fixes. If you're still stuck, bring the Streamerbot log to the Discord and someone can help.

  • Billing enabled? New API accounts need a payment method added at platform.openai.com/account/billing before the key works
  • Key copied correctly? Open C:\StreamGPT\config.py and confirm the key is there, no extra spaces or missing characters
  • Monthly limit hit? Check your usage at platform.openai.com/usage
  • Key revoked? Old keys can be deactivated — generate a new one at platform.openai.com/api-keys and update config.py

Usually one of three things:

  • The action for that command wasn't compiled — open it in Streamerbot and hit Save and Compile
  • The command trigger in Streamerbot is disabled or set to the wrong prefix
  • You opted out of that system during installation — check C:\StreamGPT to see if the relevant script folder exists
